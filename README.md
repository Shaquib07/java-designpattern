# Design Patterns Practice

> Java implementations of Gang of Four design patterns for learning and reference

## 📁 Structure

```
designPattern/
├── creational/     # Singleton, Factory, Builder, etc.
├── structural/     # Adapter, Decorator, Proxy, etc.
└── behavioral/     # Observer, Strategy, Command, etc.
```

## 🎯 Patterns Covered

### Creational
- ✅ Singleton
- ✅ Factory Method
- ⏳ Abstract Factory
- ✅ Builder
- ⏳ Prototype

### Structural
- ⏳ Adapter
- ⏳ Decorator
- ⏳ Proxy
- ⏳ Facade

### Behavioral
- ⏳ Observer
- ⏳ Strategy
- ⏳ Command
- ⏳ Template Method

*Updating as I learn...*

## Quick Start

```bash
git clone https://github.com/your-username/designPattern.git
```
Open in IntelliJ and run any pattern's `Main.java` class.
